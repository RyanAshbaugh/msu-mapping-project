# MSU Mapping Project

# Data

The data folder has the following structure

```
├── objects
│   ├── object_1
│   │   ├── combined
│   │   ├── video_1
│   │   └── video_2
│   └── object_2
│       ├── combined
│       ├── video_1
│       └── video_2
└── scenes
    ├── scene_1
    │   ├── combined
    │   ├── video_1
    │   └── video_2
    └── scene_2
        ├── combined
        ├── video_1
        └── video_2
```
